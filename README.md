# Consultor de Comunicação - Skill

Skill que aplica frameworks de 3 livros (Comece pelo Porquê, Rápido e Devagar, Comunicação Assertiva) 
diretamente ao seu contexto.

## Como usar:
- Montar comunicações e pedidos
- Tomar decisões mais claras
- Estruturar argumentos
- Analisar apresentações e receber direcionais
